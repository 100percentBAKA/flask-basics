# Flask Notes App

This Flask application allows users to store and delete notes.

## Features

- Create a new note
- View all notes
- Delete a note

## Installation
```bash
git clone https://github.com/100percentBAKA/flask-basics
```

```bash
cd flask-basics
```

```bash
pip install -r requirements.txt
```

```bash
python main.py
```

### the flask app will be up and running on the port 5000
